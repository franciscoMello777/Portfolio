Sobre Mim

Tenho 17 anos, estou no último ano do ensino médio e atualmente curso Técnico em Desenvolvimento de Sistemas. Sempre tive interesse por tecnologia e programação e, ao longo do curso, fui me apaixonando cada vez mais por essa área. Gosto de resolver problemas, aprender novas linguagens e entender como as coisas funcionam "por trás" dos sistemas. Meu objetivo é cursar Ciência da Computação para aprofundar meus conhecimentos e expandir minhas oportunidades no mercado de tecnologia.

Projetos

Correio Anônimo

Um site surpresa para uma pessoa especial!

O Correio Anônimo foi um projeto criado especialmente para surpreender uma pessoa querida. A ideia era oferecer uma plataforma onde mensagens anônimas poderiam ser enviadas de forma misteriosa e emocionante. Foi uma maneira única de expressar sentimentos e criar uma surpresa inesquecível, com total foco na privacidade e no toque pessoal.

Site dos Carros

Um site sobre o filme Carros, com informações e curiosidades.

O Site dos Carros é um projeto dedicado ao filme da Disney-Pixar, Carros. O site contém informações sobre os personagens, curiosidades, e detalhes sobre a história do filme. A ideia é proporcionar aos fãs uma plataforma completa para explorar o universo de Carros e se divertir aprendendo mais sobre os filmes e seus personagens.

Pokedex

Uma Pokédex integrada com uma API para explorar Pokémon.

A Pokedex é uma aplicação que utiliza uma API para buscar informações sobre os Pokémon. O usuário pode procurar diferentes Pokémon, ver suas habilidades, tipos e outros detalhes interessantes. O projeto visa facilitar o aprendizado sobre os Pokémon, combinando informações práticas com uma interface simples e direta.
